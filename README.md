# StreamlitMon
### Trabajo para la asignatura de Almacenamiento, Visualización y Procesamiento de Datos.
Lucía García Lado, Antonio Quijano Herrera y Alejandro Sosa Corral

Tablero alojado en https://pokestats.streamlit.app/

## Características clave
- Visualiza los detalles individuales y estadísticas de cualquier pokémon
- Filtra por generación o excluye pokémons legendarios/míticos con el menú lateral

## Cómo lanzar tablero en local
- Descarga los contenidos del respositorio
- Instala las dependencias listadas en requirements.txt

`pip install -r requirements.txt`

- Lanza el tablero desde streamlit

`streamlit run app.py`

- Accede a la dirección `localhost` mostrada en el terminal
